This is an Angular/cli Sandbox where I learn/practice concepts needed to work in the industry.
It is a test bed not production by any means

+ HTML5
+ jQuery
+ D3 Charts

  - Wheelspin
+ Animation/Video techniques
+ @ngrx/store
+ rxjs async

1. Clone the repo
2. NPM install
3. NPM start
